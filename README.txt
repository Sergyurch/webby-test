Як запустити проект?

На локальному комп'ютері мають бути встановлені PHP та MySQL.

1. Розпакуйте проект у папку (назва будь-яка) або склонуйте з репозиторію https://github.com/Sergyurch/webby-test.git
2. Імпортуйте дамп бази даних (файл webby_test.sql) на сервер MySQL.
3. У файлі project\config\connection.php пропишіть дані для доступу на сервер MySQL.
4. Відкрийте термінал та перейдіть у папку з проектом.
5. Запустіть у терміналі команду php -S localhost:8000
6. Відкрийте проект у браузері за адресою http://localhost:8000/

Проект побудований на основі MVC. У папці Core знаходяться базові класи. У папці Project знаходиться вся логіка проекту.
Щоб потрапити на сайт, потрібно пройти реєстрацію. При додаванні або імпорті, фільми перевіряються на унікальність по назві.
Додаються лише нові.
